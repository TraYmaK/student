. - любой символ
[] - поиск конкретных или диапозонных значений
[^] - поиск, исключая конкретные или диапозонные значения
^ - начало строки
$ - конец строки
\. - экранирование символа
\n - перенос строки
\d - любая цифра
\D - поиск значения, не являющегося цифрой
\s - поиск пробелов
\S - поиск значения, не являющегося пробелом
\w - поиск буквы 
\W - поиск значения, не являющегося буквой
\b - выделение границ слов
\B - выделение границ, которые не являются границами слов
\n{x} - поиск символа n x раз
\n{x, y} - поиск символа n в пределах вхождения [x, y]
\n* - поиск символа n в пределах вхождения [0, inf]
\n+ - поиск символа n в пределах вхождения [1, inf]
\n? - поиск символа n в пределах вхождения [0 / 1]
(x|y) - условие поиска or
